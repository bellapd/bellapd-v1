### How I make my personal website

# Tech Stack Used

- React
- Next.js
- ChakraUI
- MDX
- Vercel

# Introduction

Next.js is the most widely used framework for React,and the fact that Next.js takes much less time to build (only took 1~3 minutes) thus,I decided to make my own personal website with it.
There are also several tools that I used to make my website,such as Chakra UI and Vercel.

## 1. Chakra UI

Chakra is a framework that is a component-based library means It consists of fundamental building pieces that could assist you in creating the web application's front end.
Moreover, it supports `React` as well as a few other frameworks and is adaptable, reusable, and scalable code. As a rookie in website development, Chakra UI aid my development process faster.
The only thing I have to do is install and enclose every Next.js component in the `ChakraProvider` component.

## 2. Vercel

Vercel is a platform for frontend frameworks to build your static sites. It is also one of the most popular and free hosting platform for Next.js. Moreover, Vercel is simple to set up and seamlessly interacts with Github.

## 3. Next js

Next js is is a React framework for developing single page Javascript applications. It provides out-of-the-box tooling and configuration you need to build fast, SEO-friendly React apps. Next. js eliminates the need for a backend framework by taking it one step further. It allows you to create React applications that are completely server-rendered. This means that the user's browser will download a complete HTML page, including all the necessary assets like JavaScript files, CSS files, and image
